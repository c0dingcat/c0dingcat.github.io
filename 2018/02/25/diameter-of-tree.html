Diameter of tree - 트리의 직경

February 25, 2018

  • 트리의 직경은 가장 크게 떨어진 두 개의 노드 사이에 있는 노드들의 개수를 의미한다
  • 트리의 직경을 구할때 트리의 height와의 비교도 필요하다

코드

int height(node* ptr) {
    if(!ptr)
        return 0;
    return 1 + max(height(ptr->left), height(ptr->right));
}

int diameter(node *ptr) {
    
    if(!ptr)
        return 0;
    
    int lheight = height(ptr->left);
    int rheight = height(ptr->right);
    int left = diameter(ptr->left);
    int right = diameter(ptr->right);
    
    return max(lheight + rheight +1, max(left, right));   
}
