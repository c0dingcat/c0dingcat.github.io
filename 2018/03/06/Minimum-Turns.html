March 6, 2018

Minimum Turns 문제

  • 주어진 글자들은 한글자만 변형이 가능하고 맨 앞으로 옮기는 것만 허용된다
  • 문자열을 변형해서 주어진 문자열이 될떄까지의 최소 턴 수를 구하라
  • 예) 소스: abc, 타켓: cba abc->bac->cba
  • 문제가 쉬어 보이지만 어렵다… 한참을 봤는데 잘 모르겠음.

코드

int minimumTurns(const string& s1, const string& s2) {
    if(s1.size() != s2.size())
        return -1;
    
    if(s1 == s2)
        return 0;
    
    int index = s2.size() - 1;
    while(index >=0 && s1[index] == s2[index])
        index --;
    
    if(s2[index] == s1[0])
        return index;
    return index+1;
}
