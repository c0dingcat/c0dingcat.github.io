Smash Word 문제

  • To a word, and a map, map which is a word, ask this if a word is smash able? That is, you can smash one letter each time, and the rest of the letters can form a single word (the new word is still in the map) until it is completely hit. For example: sprint -> print -> pint -> pit -> it -> i -> “”

코드

bool smash_word(string& current, stack<string>& wordStack, const map<string, int>& dic) {
    int length = current.size();
    if(length == 0) {
        cout << current << endl;
        return true;
    }

    for(int i = 0 ; i < length; i++ ) {
        string word = current.erase(i);
        if(word == "" || dic.count(word)!=0) {
            wordStack.push(word);
            if(smash_word(word, wordStack, dic)) {
                return true;
            }
            else
                wordStack.pop();
        }
    }
    return false;
}
