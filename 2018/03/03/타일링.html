타일링 방법의 수 세기

March 3, 2018

출처

  • 알고리즘 문제 해결 전략 - 8.11 경우의 수와 확률

문제설명

  • 2n 크기의 사각형을 21 크기의 타일로 채우는 방법의 수를 계산하는 문제
  • 타일을 길게 붙이거나 (21) 혹은 옆으로 누워서 붙이는 경우 (2n)로 붙이는 경우를 생각해서 계산
  • 즉, f(n) = f(n-1) + f(n-2)
const int mod_num = 1000000007;

int tiling(const vector<int>& input, int width) {
    if(width <=1) return 1;
    int & ret = cache[width];
    if(ret !=-1) return ret;
    return ret = (tiling(input, width-2) + tiling(input, width-1))%mod_num;
}
