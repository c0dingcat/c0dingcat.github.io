삼각형 위의 최대 경로 개수 세기

March 3, 2018

출처

  • 알고리즘 문제 해결 전략 - 8.11 경우의 수와 확률

문제설명

  • 이전의 문제는 최대 경로의 합을 구했을뿐, 경로 자체는 구하지 않았다
  • 최대 경로는 유일하지 않을 수도 있다
  • 문제를 해결하기 위해 두 개의 다른 동적 계획법 문제를 해결
  • 합을 먼저 구하고, 그 합을 구하는 도중에 만든 캐싱을 통해 패스를 구현
struct triangle {
    vector<vector<int>> input;
    vector<vector<int>> cache;
    vector<vector<int>> countCache;
    
    int path(int y, int x) {
        if(y == input.size() -1)
            return input[y][x];
        
        int& ret = cache[y][x];
        if(ret != -1)
            return ret;
        
        return ret = max(path(y+1,x), path(y+1,x+1))+input[y][x];
    }
    
    int count(int y, int x) {
        if(y==input.size()-1)
            return 1;
        
        int & ret = countCache[y][x];
        if(ret!=-1)
            return ret;
        
        if(path(y+1,x+1)>=path(y+1,x))
            ret += path(y+1,x+1);
        if(path(y+1,x+1)<=path(y+1,x))
            ret += path(y+1,x);
        return ret;
    }
}
