우물을 기어 오르는 달팽이

March 4, 2018

문제

  • 깊이가 n인 우물이 있고, 우물의 맨 밑바닥에는 달팽이가 있다
  • 맑은 날에는 2미터를 기어 올라갈 수 있지만, 비가 내리면 1미터를 올라갈 수 있음
  • 비올 확률이 50%일때, m일 안에 달팽이가 우물 끝까지 올라갈 수 있는 확률
  • 점화식 climb(n) = f(n+1,2) + f(n+1,1)

코드

struct snail {
    int n,m;
    int vector<map<int,int>>& cache;
    int climb(int days, int climbed) {
        if(days >= m) return climbed >= n ? 1 : 0;
        int& ret = cache[days][climbed];
        if(ret != -1) return ret;
        return ret = climb(days+1, climbed +2) + climb(days+1, climbed+1);
    }
}
