트리의 PreOrder, InOrder 순회로 PotOrder 알아내기

March 4, 2018

문제

  • 주어진 PreOrder, InOrder 순환 값으로 PostOrder 순환 값을 에측하라
  • 트리를 주어진 순환 값으로 부터 구성하면서 PostOrder 순환을 한다

코드

vector<int> slice(vector<int>& src, int a, int b) {
    return vector<int>(src.begin()+a, src.end()+b);
}

void postOrder(vector<int>& preOrder, vector<int>& inOrder) {
    int length = preOrder.size();
    int root = preOrder[0]; // preOrder의 제일 처음에 나오는 값이 루트이다.
    
    int left = find(inOrder.begin(), inOrder.end(), root) - inOrder.begin();
    int right = length - left - 1; // 루트 값 제외

}
