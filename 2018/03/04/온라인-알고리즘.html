외계 신호 분석 - 온라인 알고리즘

March 4, 2018

문제

  • 알고리즘 문제해결 전력 19.6 - 외계신호 분석
  • 신호 생성 방식 A[0] = 1983, A[i] = (A[i-1]*214013+2531011) mod 232
  • 슬라이딩 윈도우 기법으로 한꺼번에 많은 메모리를 할당하지 않고, 바로 합이 k가 되는 숫자들을 구하고 있음

코드

struct asig {
    unsigned seed;
    asig():seed(1983){}
    unsinged next() {
        unsigned ret = seed;
        seed = ((seed*214013u)+2531011u);
        return ret % 1000 + 1;
    }
}

int countRanges(int k, int n) {
    asig sig_;
    queue<int> range;
    int ret = 0, rangeSum = 0;
    for(int i = 0 ;  i < n ; i++) {
        int new_sig = sig_.next();
        rangeSum += new_sig;
        range.push(new_sig);

        while(rangeSum >k){
            rangeSum -= range.front(); range.pop();
        }
        if(rangeSum == k) ret++;
    }
    return ret;
}
