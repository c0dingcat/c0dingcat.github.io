Subarray 요소의 합이 특정 숫자의 배수가 되는지를 확인

March 8, 2018

문자열 안에서 중복 문자열 찾기

  • Given a list of non-negative numbers and a target integer k,
  • write a function to check if the array has a continuous subarray of size at least 2 that sums up to the multiple of k, that is, sums up to n*k where n is also an integer. **/
bool checkSumOfSubArray(const vector<int>& input, int k) {
    for(int i = 1; i < input.size(); i++)
        input[i] += input[i-1];
    
    for(int hi = input.size() -1 ; hi >=0 ; hi--) {
        for(int lo = -1 ; lo < hi ; lo++){
            int low_value = (lo == -1)? 0 : input[lo];
            if((input[hi] - low_value)%k == 0)
                return true;
        }
    }
    return false;
}
