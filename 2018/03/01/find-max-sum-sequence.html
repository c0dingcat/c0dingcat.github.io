정수 배열에서 가장 큰 이어지는 원소들의 합을 구하라 - 시간복잡도는 O(n)

March 1, 2018

  • 출처: 매일프로그래밍 #1
  • Longest increasing subsequence
  • 이 문제는 최대 합을 구하기 위해 원소들을 계속 더해 나갈때 음의 원소가 나왔을 때 그 원소를 계속 더하는게 맞는지 아닌지를 판단하는 것이 중요하다
  • 아래에서 currentMax는 원소들을 계속 더해 가면서 현재 인덱스에서의 최대 값을 의미한다
  • maxMax 값은 이제까지 나왔던 값들 중에서 최대 값을 의미하다. 즉, 중간에 음수 값이 나왔을때 이 값이 일시적으로 currentMax보다 커질수 있다
  • 쉽지만, 헤맬수 있는 문제..
Input: [-1, 3, -1, 5]
Output: 7 ( 3 + (-1) + 5 )

Input: [-5, -3, -1]
Output: -1 ( -1 )

Input: [2, 4, -2, -3, 8]
Output: 9 ( 2 + 4 + (-2) + (-3) + 8 )

int findMaxSum(const vector<int>& input) {

    int current_sum = 0;
    int max_sum = 0 ;
    for(int i = 0 ; i < input.size(); i++) {
        current_sum = max(current_sum + input[i], input[i]);
        max_sum = max(max_sum, current_sum);
    }
    return max_sum;
}
